Seja bem vindo ao mundo da programação!

Aqui você vai aprender sobre:

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar e estruturar conteúdo em páginas da web, definindo elementos como textos, imagens e links, por meio de tags que indicam sua função e formato na página.

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para definir a apresentação e o layout de documentos HTML/XML, controlando cores, fontes, espaçamentos, posicionamento e outros aspectos visuais em páginas da web.

JavaScript é uma linguagem de programação de alto nível, interpretada, orientada a objetos e amplamente usada para criar interatividade em páginas da web, manipular elementos HTML, responder a eventos e desenvolver aplicativos web dinâmicos.
